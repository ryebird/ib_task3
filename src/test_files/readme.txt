Отлично!! Вы открыли тренировочный файл и посмотрели весь текст целиком (cat).
Команда head показывает только первые строки файла. (head -n 2).
Эта строка содержит слово важный для поиска через grep.
Команда tail выводит последние строки файла. (tail -n 2).
Команда wc -l считает количество строк в файле. (wc -l).
